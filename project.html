STIP Register 📝

About Project

STIP Register is a web-based registration system developed for the Sekolah Tinggi Ilmu Pelayaran (STIP) Jakarta to streamline the cadet admission and registration process. The application provides a centralized platform for managing participant registration, document verification, payment processing, and student data administration.

The system supports two primary user roles:

👤 Participant

Participants are required to create an account before completing their registration. Once registered, they can:

  • ✔ Complete personal and academic registration forms.
  • ✔ Upload all required supporting documents.
  • ✔ Apply for the active admission period.
  • ✔ Monitor document verification status.
  • ✔ Receive a Virtual Account (VA) code for registration payment.


👨‍💼 Administrator

The administrator manages the entire registration process through a centralized administration panel with the following modules:

  • ✔ Dashboard – Monitor overall registration activities and system status.
  • ✔ General Reference – Manage master data and general reference information.
  • ✔ Academic Reference – Configure academic-related data and registration settings.
  • ✔ Registration – Handle participant registration, document verification, and admission workflows.
  • ✔ Reports – Generate registration, payment, cadet, and academic reports.
  • ✔ System Management – Manage system configurations and operational settings.
  • ✔ User Management – Create and manage administrator accounts and access permissions.
  • ✔ Landing Page Management – Configure and update information displayed on the public website.
  • ✔ Application Settings – Manage system parameters and application configurations.
  • ✔ BNI Callback Logs – Monitor and validate payment callback transactions from the BNI Virtual Account integration.
